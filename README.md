# Aaron's Portfolio v2.0

A high-performance, developer-focused portfolio designed with a geometric Bento Grid layout and an integrated Bash-like terminal interface.

---

## Features

- **Interactive Bash Terminal:** A functional CLI handling custom logic, state-driven command history, and real-time output.
- **"Root" Directory Easter Egg:** Integrated `sudo rm -rf /` command that triggers a simulated system wipe and reboot sequence.
- **Minecraft Integration:** Custom `/gamemode` commands—a nod to my background in Minecraft server administration and Velocity proxy networking.
- **Geometric Bento Grid:** A responsive CSS Grid layout utilizing complex vertical and horizontal spanning (`row-span-2` / `col-span-2`).
- **Performance Optimized:** Built with **React** for efficient state management and **Tailwind CSS** for a low-latency, utility-first UI.
- **Cross-Platform Responsive:** Fully adaptive architecture that transitions from a 3-column technical layout to a mobile-optimized stack.

## Tech Stack

- **Frontend:** React 18
- **Styling:** Tailwind CSS
- **Environment:** Node.js / Vite

## Terminal Commands

| Command | Description |
| :--- | :--- |
| `ls skills/` | Lists technical competencies (Java, Python, Linux, etc.) |
| `cat introduction.txt` | Returns developer background and bio. |
| `clear` | Wipes the terminal buffer. |
| `/gamemode creative` | Minecraft easter egg (Flight module not included). |
| `sudo rm -rf /` | **Warning:** System-wide directory wipe sequence. |
